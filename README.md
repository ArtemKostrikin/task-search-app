# Task Search App

Поиск задач через внешний REST API с использованием Vue.js и Vuex.

## Функционал
- Поиск задач по ключевым словам
- Запросы к REST API (JSONPlaceholder)
- Управление состоянием через Vuex
- Обработка ошибок

## Стек
- Vue.js
- Vuex
- Axios

## Запуск

```bash
npm install
npm run serve
